# Aurora Market

This is the intentionally messy Expo managed brownfield fixture for `rn-mt`.

It keeps several traits that a real existing app would already have:

- custom `start:*` scripts
- `app.json` plus `app.config.ts` layering
- TypeScript path aliases
- `config/`, `theme/`, `assets/`, and `tests/` folders
- brand-specific copy that the audit flow can later flag when shared code should become tenant-specific
